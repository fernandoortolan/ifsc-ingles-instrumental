Inglês no mundo da tecnologia

É inevitável o contato com a língua inglesa nos dias atuais, principalmente quando estamos nos referindo à área da tecnologia e mercado de trabalho.

Atualmente o inglês é uma das línguas mais faladas do mundo e a língua com maior número de falantes não nativos — com quase o dobro do número de falantes nativos.

Esse é um dos motivos para que tecnologias como linguagens de programação sejam baseadas na língua inglesa. A maioria das linguagens são escritas em inglês.

Uso do inglês em linguagens como HTML e CSS

Na matéria de Desenvolvimento Web I do curso de Desenvolvimento de Sistemas, usamos tecnologias para o desenvolvimento de websites. São elas a linguagem de marcação, HTML, e a linguagem de estilização, CSS, ambas com sintaxe em inglês.

HTML é a sigla para HyperText Markup Language (Linguagem de Marcação de Hipertexto).

CSS é a sigla para Cascading Style Sheets (Folhas de Estilo em Cascata).

Em HTML usamos tags para marcar o conteúdo que será renderizado na página.

Todas as tags da linguagem são palavras ou siglas derivadas da língua inglesa, como as tags head e body.

A tag head recebe outras tags com informações gerais (metadados) sobre o documento.

Um exemplo de tag que geralmente encontramos no head é a tag title, ela contém o título da página que fica na aba do navegador.

Já a tag body, recebe tags marcando todo o conteúdo a ser renderizado pelo navegador.

Exemplos do uso do inglês nessas linguagens

Marcação de texto

Para definir um título, podemos utilizar a tag h1. A letra h é a sigla de heading. O número presente na tag representa o nível do heading, que varia de 1 a 6.

A tag mais comum utilizada para textos é a p. A letra p é a sigla de paragraph.

Divisão de conteúdo e fins semânticos

Existem tags para fins semânticos, deixando o código mais organizado e fácil de ler. Manter um código semântico ajuda em futuras alterações e na leitura do código.

Para englobar o conteúdo presente no cabeçalho da página podemos utilizar a tag header.

O conteúdo principal da página é posto, geralmente, dentro da tag main.

E no rodapé é comum utilizar a tag footer.

Outras tags como section e div, por exemplo, são usadas para divisão de conteúdo e seções dentro da página.

Listas

Para montar uma lista usamos as tags ul (unordered list) e ol (ordered list).

Dentro destas, usamos a tag li (list item), para definirmos um item da lista.

Exemplo de lista ul, não ordenada:

  • Item I
  • Item II
  • Item III

Exemplo de lista ol, ordenada:

  1. Item I
  2. Item II
  3. Item III

Tabelas

A tag table é utilizada para montar tabelas, junto com as tags tr (table row) para definir as linhas da tabela, e td (table data), para definir as colunas.

Tags com atributos

Tags como img (image) utilizada para inserção de imagens e a tag a (ancor) utilizada para definir links, possuem atributos como, src (source) e href (hypertext reference) onde é preciso informar o endereço ou diretório do arquivo de imagem, e o endereço de destino do link, respectivamente.

CSS

Após o término da marcação do conteúdo da página, utilizamos o CSS para fazer a estilização.

Para estilizar um elemento HTML, é preciso referenciá-lo de alguma forma.

Após a seleção do elemento, são definidas as regras de estilo que consistem de uma propriedade e um valor — ela será aplicada ao elemento selecionado.

Alguns exemplos de propriedade/valor:

  • position: absolute;
  • height: 128px;
  • width: 256px;
  • background-color: gray;
  • color: blue;
  • font-size: 24px;
  • font-weight: bold;
  • text-align: center;

É evidente a presença do inglês na sintaxe das linguagens mencionadas. Conhecer a língua inglesa facilita o aprendizado e colabora com a compreensão semântica no uso dessas tecnologias.

A importância do inglês como programador

Quando escrevemos um código, estamos em constante contato com o inglês, seja por conta da sintaxe das linguagens e erros que podem surgir, até referências na internet ou em documentações oficiais.

Sem dúvida o número de referências encontradas em inglês é muito maior quando vamos fazer uma pesquisa.

Pesquisando no Google por python variáveis, foram encontrados aproximadamente 437 mil resultados.

Fazendo essa mesma pesquisa em inglês, tive uma surpresa, mais de 2 bilhões de resultados encontrados, quase 5000 vezes maior que a pesquisa em português.

Porém, os números variam bastante, para mais ou para menos, de acordo com a data da pesquisa. Mesmo assim, quando pesquisamos conteúdos relacionados a tecnologia, termos em inglês geralmente possuem um resultado maior.

Sabendo inglês, consequentemente, teremos acesso a mais materiais e conteúdos disponíveis para consulta e pesquisa.

Escrevendo códigos em inglês

Embora a sintaxe de uma linguagem de programação seja em inglês, ainda podemos optar por nomear variáveis e funções em português. Porém, isso dificulta programadores não falantes da língua portuguesa a entender o código.

Hoje em dia, muitas empresas já adotaram como padrão escrever códigos apenas em inglês, evitando problemas de comunicação e organização entre os desenvolvedores.

Lidando com erros

Na programação, quando nos deparamos com um erro, ele também será apresentado em inglês. Tendo um breve conhecimento da língua inglesa, fica muito mais fácil entender o erro e achar uma solução.

Oportunidades no exterior

Hoje existem muitas vagas no exterior na área da tecnologia da informação — remotas e presenciais.

O inglês é uma ferramenta indispensável quando se trata dessas vagas — mesmo em países falantes da língua portuguesa — pois empresas que atuam internacionalmente contratam pessoas do mundo todo.
